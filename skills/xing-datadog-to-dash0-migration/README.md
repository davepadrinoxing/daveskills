# What this skill covers:

- Odyssey/Enclave deploy-shape discovery.
- Datadog usage discovery.
- Standard odyssey-helm-app Dash0 baseline.
- “Only one release enables dash0.enabled” rule.
- Prometheus applicability check instead of assuming it.
- Browser RUM as conditional/follow-up unless tokens are ready.
- Deploy-time Job immutability issue and dash0.com/enable: "false" mitigation.
- SRE/SLO checks, including Datadog-only leftovers.
- Preview validation checklist for Argo, Dash0 traces/logs/metrics, workers, errors.
- Completion criteria for deciding when the migration is done.

The skill is intentionally conservative: it helps assess and plan migrations safely, and only proceeds to implementation when repo context and missing inputs are clear.
